# DSpamerV2
D-Spamer version 2
## Termux
Установка проста
клонируете репозиторий 
* git clone https://github.com/defsec16/DSpamerV2.git
Входите в папку
* cd DSpamerV2
* python DSpamerV2.py
* pkg install git и python если не установлен
## Linux
apt-get install python или python3
apt-get install git
git clone https://github.com/defsec16/DSpamerV2.git
cd DSpamerV2
python DSpamerV2.py
